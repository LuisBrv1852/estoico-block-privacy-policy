Política de Privacidad de Estoico Block
Última actualización: 05/05/2026
Estoico Block es una aplicación diseñada para ayudar a los usuarios a reducir el acceso impulsivo a contenido para adultos o contenido sensible mediante un sistema de bloqueo basado en palabras clave y una pantalla de pausa con ejercicios de respiración.
Esta Política de Privacidad explica qué información puede ser accedida por la aplicación, cómo se utiliza, cómo se conserva y cómo puede eliminarse.
1. Información que accede la aplicación
Estoico Block puede acceder localmente a cierta información del dispositivo para cumplir con su función principal de bloqueo de contenido. Esta información incluye:
	• Texto visible en pantalla dentro de aplicaciones seleccionadas. 
	• Descripciones de elementos visibles en pantalla. 
	• Nombre de la aplicación activa. 
	• Palabras clave configuradas para el bloqueo. 
	• Preferencias locales de configuración de la aplicación. 
La aplicación no solicita ni recopila datos como nombre, edad, dirección, número telefónico, contactos, fotografías, ubicación precisa, información bancaria o credenciales de acceso.
2. Uso del Servicio de Accesibilidad
Estoico Block utiliza el Servicio de Accesibilidad de Android para detectar texto visible en pantalla dentro de aplicaciones seleccionadas por el usuario, como navegadores y redes sociales.
Este permiso es necesario para que la aplicación pueda comparar el texto visible con palabras clave relacionadas con contenido para adultos o contenido sensible. Si se detecta una coincidencia, Estoico Block muestra una pantalla de bloqueo temporal con un ejercicio de respiración.
La aplicación no utiliza el Servicio de Accesibilidad para:
	• Espiar conversaciones. 
	• Robar información personal. 
	• Realizar compras. 
	• Enviar mensajes. 
	• Controlar otras aplicaciones. 
	• Automatizar acciones en nombre del usuario. 
	• Compartir texto visible con terceros. 
El análisis del texto se realiza localmente en el dispositivo.
3. Cómo se usa la información
La información accedida mediante el Servicio de Accesibilidad se usa únicamente para:
	• Detectar coincidencias con palabras clave configuradas. 
	• Identificar contenido sensible o para adultos. 
	• Mostrar una pantalla de bloqueo temporal. 
	• Aplicar preferencias de uso configuradas por el usuario. 
	• Mejorar la experiencia de autocontrol digital dentro de la aplicación. 
Estoico Block no vende, comparte ni transfiere a terceros el texto detectado mediante el Servicio de Accesibilidad.
4. Almacenamiento y conservación de datos
Estoico Block no almacena en servidores externos el texto visible detectado en pantalla.
Las preferencias locales de la aplicación, como configuración de bloqueo, estado de protección o palabras clave, pueden guardarse localmente en el dispositivo del usuario.
Estos datos locales permanecen en el dispositivo hasta que el usuario los modifique, elimine los datos de la aplicación o desinstale Estoico Block.
5. Eliminación de datos
El usuario puede eliminar los datos locales de Estoico Block de las siguientes formas:
	• Borrando los datos de la aplicación desde los ajustes de Android. 
	• Desinstalando la aplicación. 
	• Modificando o eliminando las palabras clave configuradas dentro de la aplicación, si esta función está disponible. 
	• Desactivando el Servicio de Accesibilidad desde los ajustes del sistema Android. 
Como Estoico Block no almacena datos personales en servidores externos, no existe una cuenta en línea asociada a la aplicación ni datos remotos que eliminar.
6. Compartición de datos
Estoico Block no vende, comparte ni transfiere a terceros la información detectada mediante el Servicio de Accesibilidad.
La aplicación no comparte texto visible en pantalla, palabras clave detectadas ni información de uso sensible con anunciantes, redes publicitarias u otros servicios externos.
7. Seguridad
La información utilizada por Estoico Block se procesa localmente en el dispositivo del usuario. La aplicación no transmite a servidores externos el texto detectado mediante el Servicio de Accesibilidad.
Recomendamos mantener el dispositivo actualizado y descargar la aplicación únicamente desde fuentes confiables.
8. Menores de edad
Estoico Block no está dirigida específicamente a niños. Si un menor utiliza la aplicación, debe hacerlo bajo supervisión de su padre, madre o tutor legal.
9. Cambios en esta Política de Privacidad
Podemos actualizar esta Política de Privacidad cuando sea necesario para reflejar cambios en la aplicación, requisitos legales o políticas de distribución.
Cualquier cambio será publicado en esta misma página.
